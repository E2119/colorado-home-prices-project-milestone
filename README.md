# colorado-home-prices-project-milestone
Colorado Average home Prices (2000–2025) – ML Project Milestone

This project explores Colorado’s average home prices from 2000–2025 using the Zillow Home Value Index (ZHVI) dataset.

        What this milestone includes

Loaded the ZHVI dataset (2000–2025)

Cleaned the Date column and selected Colorado’s price data

Converted monthly values into yearly averages

Created a line plot showing the price trend

Added a short interpretation of the trend

      Goal of the full project

The final project will use Python + Pandas + simple ML models to understand and predict Colorado housing price trends.
